# house-price-prediction
Machine Learning project to predict house prices using Random Forest
## Problem Statement:
This project aims to predict house prices using machine learning techniques based on various features.
##Dataset:
The dataset contains housing-related features such as crime rate, number of rooms, and other attributes.
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
## Model Used
Random Forest Regressor

## Evaluation Metrics
- MAE
- MSE
- R² Score

## Visualization
Used Matplotlib to visualize Actual vs Predicted values.

## How to Run
1. Open the notebook in Jupyter or Google Colab
2. Run all cells

## Conclusion
The model predicts housing prices with good accuracy using Random Forest.
